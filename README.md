# 🎙️ Kavya AI - Voice Assistant

Kavya AI is a Python-based voice assistant that listens to your voice commands and performs tasks like opening apps, playing music, searching the web, and telling time or date.

## 🚀 Features

- 🎧 Voice recognition using SpeechRecognition  
- 🗣️ Text-to-speech using pyttsx3  
- 🌐 Open websites and applications  
- 🎵 Play songs, movies, and shows  
- 🔍 Perform Google search  
- ⏰ Tell current time and date  
- 🧠 Easy to extend with new commands  

## 🛠️ Technologies Used

- Python  
- SpeechRecognition  
- pyttsx3  
- sounddevice  
- webbrowser  

## ▶️ How It Works

1. The assistant listens for the wake word **"Kavya"**  
2. After activation, it listens for your command  
3. Processes the command and performs the task  
4. Responds using voice output
